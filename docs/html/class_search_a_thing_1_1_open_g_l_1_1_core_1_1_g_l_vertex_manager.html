GLVertexStruct[] GetVboArray ()
 Used internally by the SearchAThing.OpenGL.Render.GLModel.RenderVertexManager to render the scene.

Contains the list of vertex structs directly mapped to the gpu.
This array will be built each time a vertex changes from the ToArray() of the list GLVertexManager.Vertexes. To avoid performance penalty in a scenario where you need to change the position of a figure you can use IGLFigure.ObjectMatrix that doesn't affects at all vertexes because is a figure related information forwarded to the gl pipeline shader through SearchAThing.OpenGL.Render.Constants.UNINAME_uObject.

bool ExpandModelBBox [get]
 Flag to state if vertex added to this vertex manager imply expansion of the gl model (Default: true).
More (custom) vertex manager can be registered into the gl model, may for temporary purpose and setting this variable to false at the construct of these custom vertex manager allow to unaffect the gl model bbox.
Should be used to avoid the zoomfit to evaluate temporary vertex manager figures.
