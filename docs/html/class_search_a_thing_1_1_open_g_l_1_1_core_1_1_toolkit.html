static

Retrieve color that is in the given offset in given hue range.
Default: hue range [240, 300] generating gradients in offeset range [0, 1] traversing following colors ( [blue, cyan, green, yellow, orange, red, fuchsia].

Following some colors hue values:

  • 240 = Blue (0,0,255)
  • 180 = Cyan (0,255,255)
  • 120 = Green (0,255,0)
  • 60 = Yellow (255,255,0)
  • 30 = Orange (255,128,0)
  • 0 = Red (255,0,0)
  • 300 = Fuchsia (255,0,255)
  • 270 = Violet (128,0,255)

    ref: https://en.wikipedia.org/wiki/HSL_and_HSV

    Parameters
    offsetOffset in range [0,1] of the color to choose in given hue range.
    hueFromHue of color range from.
    hueToHue of color range to.
    decreaseHueIf true (default) offset traverse hue range decreasing hue starting by "from" toward "to". Otherwise "to" hue will reached by the "from" through increment of hue.
    Returns
    Color corresponding given range offset.
