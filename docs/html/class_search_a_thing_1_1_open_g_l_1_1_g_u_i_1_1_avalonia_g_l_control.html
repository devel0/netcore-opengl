Detailed Description

Avalonia gl control.
It's a graphical ui IRenderDevice used to transfer GLControl pixels to the window bitmap through the override of the avalonia Render method that will be called by the avalonia framework every time the visual invalidates.
When control size changed GLControl.Invalidate will be invoked and GLSizeChanged event emitted.

Mouse wheel changes are handlded through GLControl.CameraZoomIn and GLControl.CameraZoomOut.
Mouse middle button press can start pan through GLControl.Pan.
Mouse middle double click activate GLControl.ZoomFit.
Mouse left pointer start model rotate through GLControl.ModelRotate.
